stewie

This project is result of FedexDay at Globo.com.

The main idea of the project is help you on the detection of anomaly in a generic mass of data. It receives and stores generic parameters and learns how they are correlated. If some event occurs and these parameters exhibit a strange behavior, it alerts.

Since we have a timebox to finish this project, more documentation about how the things work will be posted here in the next days. Basically, we used some theory from here.
